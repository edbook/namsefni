4. Kafli

Það getur verið gagnlegt að skoða einnig má skoða kóðann á bak við þær til að sjá hvernig hitt og þetta er gert. Kóðann má finna á https://github.com/edbook/haskoli-islands/tree/main/projects/tmp001g en einnig er hægt að smella á Edit on GitHub efst til hægri. Þá opnast viðkomandi síða á GitHub og til að sjá kóðann á bak við hana er smellt á Raw.
